O rozšíření

Web-ext-slovnikcz je jednoduché rozšíření kontextového menu prohlížečů Chrome a Firefox, které zkusí přeložit vybrané slovo pomocí serveru http://www.slovnik.cz/. Vyskočí malý dialog s překladem, který lze zrušit buď kliknutím kamkoli ve stránce nebo stiskem klávesy escape.

Pro všechny kterým lištička od seznamu nevyhovuje.

V současné chvíli podporuje jen překlad z angličtiny do češtiny!

Instalace

Chrome: Nejlepé si rozšíření nainstalujte přímo z chrome webstoru.

Firefox 57+: : Nejsnažší je instalace přímo z Firefox addons.

Podpora

Prosím pro hlašení chyb, návrhů nebo čehokoli dalšího použijte chrome webstore podporu nebo přímo github issues.
