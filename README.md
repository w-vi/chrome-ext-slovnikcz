# web-ext-slovnikcz

This is a very simple Webextension to translate a selected word using the http://www.slovnik.cz/ site.

It pops up a modal window with translation from http://www.slovnik.cz/ to get rid of it just click the mouse or press escape.

Install it directly from the [chrome webstore](https://chrome.google.com/webstore/detail/p%C5%99elo%C5%BEit-pomoc%C3%AD-slovnikcz/lhibffeamhlmgdhohnjcnlckebkkdjim)

Or for Firefox on the [mozilla addons](https://addons.mozilla.org/en-US/firefox/addon/prelozit-pomoci-slovnik-cz/).

**It supports only english to czech translation at the moment**
